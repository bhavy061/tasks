# Task Manager Board

A clean and responsive Kanban-style Task Manager built using **HTML, CSS, and Vanilla JavaScript**.
This project allows users to create, edit, move, and delete tasks with an elegant modern UI inspired by productivity tools like Notion and Trello.

---

## ✨ Features

* 📌 Create new tasks
* ✏️ Edit existing tasks
* 🗑️ Delete tasks
* 🔄 Move tasks between stages
* 🎯 Priority labels (High, Medium, Low)
* 🌙 Automatic Dark Mode support
* 📱 Fully responsive design
* ⚡ Smooth animations and toast notifications
* 🧾 Modal-based task management UI
* 🔐 Safe HTML rendering to prevent injection

## 📋 Task Workflow

Tasks can move through three stages:

1. **Todo**
2. **In Progress**
3. **Done**

Each task includes:

* Title
* Description
* Priority level
* Status/stage
